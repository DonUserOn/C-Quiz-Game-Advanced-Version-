# 🎯 C++ Quiz Game (Advanced Version)

## 📌 Overview

This project is an advanced console-based Quiz Game developed in C++. It demonstrates strong fundamentals of programming, clean code structure, and the use of modern C++ features.

The application presents multiple-choice questions to the user, evaluates answers, and calculates the final score with a percentage result.

---

## 🚀 Features

* 🧠 Multiple-choice quiz system
* 🎲 Randomized question order (every run is different)
* ✅ Real-time answer validation
* 📊 Score and percentage calculation
* 🧩 Modular and scalable code design using `struct` and functions
* ⚡ Clean and user-friendly console interface

---

## 🛠️ Technologies Used

* C++ (Standard Library)
* STL Containers (`vector`)
* Algorithms (`shuffle`)
* Structured Programming & Functions

---

## 📂 Project Structure

* `main.cpp` → Core application logic
* Question system implemented using `struct` for scalability

---

## ▶️ How to Run

### 1. Compile

```bash
g++ main.cpp -o quiz
```

### 2. Run

```bash
./quiz
```
